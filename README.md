# Guess the Number (Java) ////////// Тестовая версия пулл реквеста

Консольная игра «Угадай число».

## Описание
Компьютер загадывает число от 1 до 100.
Пользователь пытается угадать его, получая подсказки «больше» или «меньше».
Ведётся подсчёт количества попыток.
Лучший результат (рекорд) сохраняется в файл.

## Функциональность
- угадывание числа
- повтор игры
- сохранение рекорда в файл

## Запуск
1. Открыть проект в IDE
2. Запустить класс `Main`
